# 7.4 工程变更价款确定

## 1 监理对变更的管理

* 总监组织专监审查、评估费用和工期影响
* 总监组织甲乙双方协商，共签变更单
* 监理督促乙方按变更后施工

## 2 工程变更价款确定方法

### (1) 已标价清单项目

* 如有能适用的，直接适用（偏差15%以上适当调整）
* 没有能适用的，但有类似的，参照
* 没有类似的，乙方提出，报甲方确认

### (2) 措施项目费

* 安全文明施工费不得浮动
* 单价计算：按实际变化项目确定
* 总价计算：按浮动率计算

{% hint style="info" %}
报价浮动率

* 招标：1-中标价÷招标控制价
* 非招标：1-报价值÷施工图预算
{% endhint %}
